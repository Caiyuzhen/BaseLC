## Install
pip3 install -r requirements.txt
- pip3 install langchain
- pip3 install openai
- pip3 install numexpr
- pip3 install pypdf
- pip3 install tiktoken 
  - OpenAI 的向量标记依赖
- pip3 install chromadb 
  - 本地向量存储解决方案

- export PYTHONIOENCODING=utf-8
